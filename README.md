# 🎵 Moody Player

Moody Player is a mood-based music recommendation web application that suggests songs according to the user's current mood. The application provides a simple and interactive interface where users can select or detect their mood and instantly receive music recommendations that match their emotions.

## 🚀 Features

- 😊 Mood-based song recommendations
- 🎧 Music playback controls
- 🔍 Search songs by title
- 📂 Categorized playlists
- 🌙 Responsive and modern UI
- ❤️ Favorite songs functionality
- 📱 Mobile-friendly design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/moody-player.git
cd moody-player
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

### Start Backend Server

```bash
npm start
```

### Start Frontend

```bash
cd ../client
npm start
```

Application will run at:

```
http://localhost:3000
```

## 🎯 How It Works

1. User selects a mood (Happy, Sad, Relaxed, Energetic, etc.).
2. The application processes the mood input.
3. Matching songs are fetched from the database.
4. Recommended songs are displayed.
5. User can play and manage songs directly from the player.
